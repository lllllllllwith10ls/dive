Dive is a Threes-like game blending additive and multiplicative number theory. Tiles merge with their divisors.

How to play: Use your arrow keys or swipe to move the tiles (WASD and HJKL work too). When two tiles touch, one of which divides the other, they merge into one, adding their values!

When you make a new tile N, the game divides N by all current seeds, and the result is "unlocked" as a new seed if it is not a unit. These new seeds aren't actually always prime! (Can you get a composite one?)

This game is set in the ring of Z[√2], the set of numbers a + b√2 where a and b are integers. Not all primes in Z are primes in Z[√2].
